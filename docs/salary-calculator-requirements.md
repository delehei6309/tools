# 薪资扣税计算系统需求文档

## 1. 项目概述

### 1.1 项目名称
薪资扣税计算系统（实发薪资计算器）

### 1.2 项目背景
用户每月税前收入固定，需要一个便捷的工具来计算每月实发工资和应缴纳的个人所得税。

### 1.3 项目目标
开发一个基于中国大陆税收政策的薪资计算系统，支持输入税前工资及各项扣除项，自动计算每月实发工资和全年个人所得税。

---

## 2. 功能需求

### 2.1 基础信息录入

#### 2.1.1 税前工资
- 支持输入月度应发工资（税前）

#### 2.1.2 社保公积金扣除
支持按项目细分录入以下扣除项：
| 扣除项 | 说明 |
|--------|------|
| 养老保险 | 个人缴纳部分 |
| 失业保险 | 个人缴纳部分 |
| 医疗保险 | 个人缴纳部分 |
| 住房公积金 | 个人缴纳部分 |

#### 2.1.3 专项附加扣除
支持按项目细分录入以下扣除项：
| 扣除项 | 说明 |
|--------|------|
| 住房贷款利息 | 首套房贷款利息支出 |
| 3岁以下婴幼儿照护 | 子女照护费用扣除 |

> **扩展预留**：后续可增加其他专项附加扣除项（如子女教育、继续教育、大病医疗、住房租金、赡养老人等）

### 2.2 年度选择
- 支持选择计算年度
- 系统根据所选年度自动适配对应的中国大陆个税政策

### 2.3 计算功能

#### 2.3.1 月度计算
- 计算每月应纳税所得额
- 计算每月应缴个人所得税（累计预扣法）
- 计算每月实发工资

#### 2.3.2 年度汇总
- 计算全年累计应纳税所得额
- 计算全年应缴纳个人所得税总额
- 展示每月实发工资明细

---

## 3. 计算规则

### 3.1 个税计算公式
```
应纳税所得额 = 税前工资 - 起征点(5000元) - 社保公积金扣除 - 专项附加扣除
应纳个人所得税 = 应纳税所得额 × 适用税率 - 速算扣除数
实发工资 = 税前工资 - 社保公积金扣除 - 应纳个人所得税
```

### 3.2 个人所得税税率表（综合所得适用）
| 级数 | 全年应纳税所得额 | 税率 | 速算扣除数 |
|------|------------------|------|------------|
| 1 | 不超过36,000元 | 3% | 0 |
| 2 | 超过36,000元至144,000元 | 10% | 2,520 |
| 3 | 超过144,000元至300,000元 | 20% | 16,920 |
| 4 | 超过300,000元至420,000元 | 25% | 31,920 |
| 5 | 超过420,000元至660,000元 | 30% | 52,920 |
| 6 | 超过660,000元至960,000元 | 35% | 85,920 |
| 7 | 超过960,000元 | 45% | 181,920 |

### 3.3 累计预扣法
采用累计预扣法计算每月个税：
```
本期应预扣预缴税额 = (累计预扣预缴应纳税所得额 × 预扣率 - 速算扣除数) - 累计减免税额 - 累计已预扣预缴税额
```

## 5. 非功能需求

### 5.1 易用性
- 界面简洁直观，操作便捷
- 支持一次录入，自动计算全年数据

### 5.2 准确性
- 计算结果精确到分（小数点后两位）
- 税率政策数据需与官方保持一致

### 5.3 可维护性
- 税率表支持配置化，便于政策更新时调整
